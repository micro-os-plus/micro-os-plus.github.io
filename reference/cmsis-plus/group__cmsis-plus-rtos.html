Detailed Description

CMSIS++ RTOS C++ API summary page.

The C++ API is the main interface to the CMSIS++ RTOS, and is the native interface used for the implementation.

The API is composed of a series of classes, subclasses, types and several namespaces.

To use the C++ API, include the <cmsis-plus/rtos/os.h> header.
