Overview

This site contains detailed reference documentation for the third edition of the µOS++ GitHub project.

With most functions implementing the POSIX semantics, but written in C++, these specifications can also be seen as a C++ version of some POSIX parts, thus the name POSIX++.

For tutorials and general documentation, check the main project web site.
